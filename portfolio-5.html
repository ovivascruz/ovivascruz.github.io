Plataforma web PCNAT

Descripción

  • Centro Nacional de Alerta de Tsunami de la MGP

  • dhn.mil.pe/cnat

  • 2021

Plataforma web diseñada para registrar y evaluar eventos sísmicos en el Océano Pacífico, generando boletines de Sismo-Tsunami que se distribuyen a través de diversos canales de comunicación (app móvil "MGP Tsunamis", SMS, correo electrónico, página web, Facebook, Twitter y WhatsApp).

Tecnologías

  • PHP Laravel
  • SQL Server
  • Javascript
  • Vue JS
